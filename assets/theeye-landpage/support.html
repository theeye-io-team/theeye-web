How to install theeye agent and get it running. #1

Create your account (at this time only accounts invitatees are allowed, contact us). Copy your agent install url. Run at your shell. When Spock tells you re done, well you are done

Task Creation /Edition / Passing Arguments to Scripts #2

  • 1- After login go to Tasks Administration
  • 2- Create a new task, give it a name, create a new script.
  • 3- Paste this gist for a hello world with arguments script and submit it.
  • 4- Add Arguments at 'Script Arguments' if you need to, e.g for execute shellScript var1 just add var1 as $1 argument.
  • 5- Select the host where that task would run.

Monitoring Creation / Types / Edition /Thresholds #3

  • 1- After login go to Monitors Administration
  • 2- Create a new monitor, You have different types.
  • 2.a- Web checks, This monitor search for a text on the response for a given URL.
  • 2.b- Process, This monitor verify if a process is running on a selected host.
  • 2.c- You create a script monitor, where if the script execution returns failure as an echo Well your monitor would fail.

Templates Creation / Types / Edition /Thresholds #4

Templates allow us to create groups based on hostnames regular expressions and assign them tasks and monitors by default so If your infrastructure scales horizontal new servers that mach for example nodep expresion Would recieve tasks/monitor defined for nodep group.
  • 1- Go to Templates Administration
  • 2- +New template
    • 3- Define your hostnames regular expression
  • 4- Add Tasks / Monitors to That group.
